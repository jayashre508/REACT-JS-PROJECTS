# 🚀 TrackFlow

### Intelligent Engineering Management Platform

TrackFlow is a modern full-stack engineering management platform designed to simplify agile software development by centralizing sprint planning, backlog management, defect tracking, project collaboration, and engineering insights into a unified workspace.

Built with a scalable React frontend and Express backend, TrackFlow enables development teams to organize work efficiently, improve project visibility, and streamline software delivery through interactive dashboards, real-time collaboration, and intelligent workflow management.

## Key Features

* Sprint Planning & Backlog Management
* Kanban Task Management
* Defect & Issue Tracking
* Engineering Insights Dashboard
* Intelligent Sprint Risk Analysis
* Duplicate Issue Detection
* Release Notes Generation
* Smart Task Search
* Real-time Collaboration with Socket.IO
* JWT Authentication
* File Attachments
* Responsive Design

## Technology Stack

* React 19
* Vite
* Tailwind CSS
* Zustand
* Express.js
* SQLite
* Socket.IO
* Multer
* Nodemailer
* Recharts

## Installation

```bash
npm install
npm run dev
```

TrackFlow demonstrates modern engineering management workflows through scalable architecture, reusable components, responsive UI, and full-stack application development.
