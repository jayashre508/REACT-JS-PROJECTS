# 👥 TalentLens

### Hiring Operations Platform

TalentLens is a modern recruitment operations platform that centralizes candidate management, interview workflows, recruiter collaboration, hiring analytics, and recruitment reporting into a single administrative dashboard.

Designed for recruitment teams and hiring managers, the platform streamlines every stage of the hiring lifecycle while improving visibility into recruitment performance through interactive dashboards, structured workflows, and centralized candidate management.

## Key Features

* Candidate Management
* Recruitment Pipeline
* Interview Scheduling
* Interview Feedback
* Recruiter Dashboard
* Hiring Analytics
* Offer Management
* Resume Management
* Advanced Search & Filtering
* Role-Based Access Control
* Responsive Dashboard

## Technology Stack

* React
* Refine
* Material UI
* React Query
* REST APIs
* React Hook Form

## Installation

```bash
npm install
npm run dev
```

TalentLens showcases scalable admin architecture, reusable UI components, responsive dashboards, advanced data management, and enterprise recruitment workflows.
