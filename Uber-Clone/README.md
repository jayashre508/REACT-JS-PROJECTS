# Uber Clone

A full-stack ride booking mobile application inspired by Uber, built to explore real-world mobile application architecture, authentication, payments, maps integration, and ride management.

---

## Features

- User Authentication
- Google Sign In
- Live Location Tracking
- Google Maps Integration
- Ride Booking
- Driver Selection
- Ride History
- Stripe Payment Integration
- Profile Management
- Responsive Mobile UI

---

## Tech Stack

Frontend

- React Native
- Expo
- TypeScript
- Tailwind CSS (NativeWind)
- Zustand

Backend

- PostgreSQL
- Serverless APIs
- Clerk Authentication

Third Party Services

- Google Maps API
- Stripe
- Geoapify

---

## Installation

```bash
git clone <your repo>

cd uber-clone

npm install

npx expo start
```
